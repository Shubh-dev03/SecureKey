# SecureKey

SecureKey is a lightweight web application for generating secure passwords and memorable passphrases with customizable security options. 
It helps users create stronger credentials quickly with real-time strength analysis, copy-to-clipboard functionality, and a clean responsive interface.

## Live Demo

https://secure-key-rho.vercel.app/

##  Features

- Generate secure random passwords
- Generate memorable passphrases
- Adjustable password length
- Uppercase, lowercase, numbers, and symbols support
- Exclude similar characters
- Exclude ambiguous symbols
- Password strength meter
- Copy to clipboard with one click
- Password history
- Dark mode support
- Responsive design for desktop and mobile

## Why I Built This

While working on web projects and managing multiple accounts during development and testing, I frequently needed secure passwords. 
Existing tools often felt cluttered or overloaded with ads, so I built SecureKey to provide a simpler and cleaner experience.
